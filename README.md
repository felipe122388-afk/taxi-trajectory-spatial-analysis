- **Installer won’t run:** Make sure your Windows is updated. Restart and try again.

- **Data import fails:** Check that your data uses UTF-8 encoding and the correct file format.

## 📖 About the Data and Technology

This tool uses PostgreSQL, a popular database system known for storing large amounts of data safely.

PostGIS extends PostgreSQL. It lets the database understand map data, such as points, lines, and areas.

By combining these, the program can quickly find spatial patterns in large taxi datasets.
